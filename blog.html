What is the Document Object Model (DOM)?
Document Object Model is the data representation of a website components. It represent elements in a structured way.
How do you select an element from the DOM?
We can select an element in multiple way. We can use element name, class name, id and attribute to select an element.

It is event handling system. Normally event bubbles from clid elements to higher element. When we want to handle a event based on it parent element then it calls event delegation and It is helpful method.
